### London Rooms
Este pequeno projeto é baseado no Airbnb.
Foi o segundo desafio do [Hiring Coders](https://hiringcoders.gama.academy/).

Pacotes utilizados:

* [Flatpicker](https://flatpickr.js.org/)
* [Leaflet](https://leafletjs.com/)

Todo o resto foi feito do zero.

Demo disponível [aqui.](https://london-rooms.netlify.app/)
